SEOmoz currently offers two core pricing options for the Mozscape API: the free version and the paid version. Our free version is great when you need a relatively small amount of information about a handful of different pages. The paid version is designed to let you drill down deep into a site, page or link. Available Free URL Metrics Calls

Available Paid URL Metrics Calls

Root Domain mozTrust
The mozTrust of the root domain of the URL
External mozRank
The portion of the URL's mozRank coming from external links
Subdomain External Domain Juice
The portion of the mozRank of all pages on the subdomain coming from external links
Root Domain External Domain Juice
The portion of the mozRank of all pages on the root domain coming from external links
Subdomain Domain Juice
The mozRank of all pages on the subdomain combined
Root Domain Domain Juice
The mozRank of all pages on the root domain combined
HTTP Status Code
The HTTP status code recorded by Mozscape for the URL
Links to Subdomain
Total links (including internal and nofollow links) to the subdomain of the URL
Links to Root Domain
Total links (including internal and nofollow links) to the root domain of the URL
Root Domains Linking to Subdomain
The number of root domains with at least one link to the subdomain of the URL
Page Authority
A score out of 100-points representing the likelihood for arbitrary content to rank on this page
Domain Authority
A score out of 100-points representing the likelihood for arbitrary content to rank on this domain
All External Links Page to Page
The number of external links from one page to another (included followed and nofollowed)
Followed Domains Linking Page
The number of unique domains with followed links to the target URL
Followed IPs Linking
The number unique IPs with a followable link to a target URL
IPs Linking
The total number of unique IPs linking to a target URL
Followed Domains to Page
The number of unique domains with followed links to a given URL
All Cblock Linking
The total number unique cblocks linking to a page
Followed CBLocks Linking
The total number unique cblocks with followed links to a page
Followed Subdomain Linking Domains
A count of all unique subdomains with followed links to the target domain
Subdomain External Links
The total number (followed and nofollowed) external links to the subdomain of the URL
Followed Subdomain Subdomains Links
The number of subdomains with followed links to the subdomain of the URL
Followed Domain Subdomains Links
The number of unique domains with followed links to the subdomain of the URL
Followed Root Domain Links
The total number of followed links (both internal and external) from a page to a domain
All Root Domain External Links
The total number of external links (both followed and no-followed) from a page to a domain
All Followed Root Domains Linking Domain
The total number of followed root domains linking to the target's domain
IPs Linking to Domain
The total number of unique IPs linking to the target's domain
Followed IPs Linking to Domain
The total number of unique IPs with followed links to the target's domain
All Cblock Linking Domain
The number of unique cblocks with a link to a domain
Followed Cblock Linking Domain
The total number of cblock with followed links to a domain

Available Paid Links Calls

Page to Page
Returns a set of Source pages linking to the specified Target page
Page to Subdomain
Returns a set of Source pages linking to any page on the specified target's subdomain
Page to Domain
Returns a set of source pages linking to any page on the specified Target's root domain
Domain to Page
Returns a single link from a set of root domains linking to the target's specified page
Domain to Subdomain
Returns a single link each from a set of root domains linking to any page on the specified target's subdomain
Domain to Domain
Returns a single link each from a set of root domains linking to any page on the specified target's root domain

Top Pages Call

Returns a list of the top pages for a given subdomain. Metrics can also be requested for each page.

Available Paid Anchor Text Calls

Term or Phrase
The anchor text term or phrase
Internal Pages Linking
The number of internal pages linking with this term or phrase
Internal Subdomains Linking
The number of subdomains on the same root domain with at least one link with this term or phrase
External Pages Linking
The number of external pages linking with this term or phrase
External Subdomains Linking
The number of external subdomains with at least one link with this term or phrase
External Root Domains Linking
The number of (external) root domains with at least one link with this term or phrase
Internal mozRank Passed
The amount of mozRank passed over all internal links with this term or phrase
External mozRank Passed
The amount of mozRank passed over all external links with this term or phrase
Flags
A bit flag column
